Class Slideshow

A simple YUI3 slideshow kit inspired by the jQuery Cycle plugin.

Constructor

Slideshow ( config )
Parameters:
config <Object> Widget configuration object.

Properties

NAME - String

Default Value: Slideshow

PRESETS.fade - Preset Slide Transition

Incoming slide fades in. Outgoing slide fades out.

PRESETS.slideDown - Preset Slide Transition

Both slides slide from top to bottom.

PRESETS.slideLeft - Preset Slide Transition

Both slides slide from right to left.

PRESETS.slideRight - Preset Slide Transition

Both slides slide from left to right.

PRESETS.slideUp - Preset Slide Transition

Both slides slide from bottom to top.

Configuration Attributes

_incomingSlide - protected Node Instance

Incoming slide Node.
Default Value: null

_isHoverPaused - protected Boolean

Hover paused flag.
Default Value: false

_isPaused - protected Boolean

Paused flag.
Default Value: true

_outgoingSlide - protected Node Instance

Outgoing slide Node.
Default Value: null

_timeoutID - protected Integer

The ID returned by autoplay's setTimeout call.
Default Value: null

autoplay - Boolean

Set to false to disable autoplay.
Default Value: true

currentIndex - Integer

Current slide index.
Default Value: 0

duration - Float

Default duration for slide transitions.
Default Value: null

easing - String

Default slide transition easing.
Default Value: ease-out

interval - Float|Array

Time interval(s) between slide transitions in seconds.
Default Value: 4

nextButton - Mixed

Next slide button Node/NodeList/selector string.
Default Value: null

pages - Mixed

NodeList/selector string for the elements that correspond to slides. This setting works best with list items.
Default Value: null

pauseButton - Mixed

Pause button Node/NodeList/selector string.
Default Value: null

pauseOnChange - Boolean

Set to false to continue autoplay when the user changes slides.
Default Value: true

pauseOnHover - Boolean

Set to false to continue playing when the user hovers over the slideshow.
Default Value: true

playButton - Mixed

Play button Node/NodeList/selector string.
Default Value: null

previousButton - Mixed

Previous slide button Node/NodeList/selector string.
Default Value: null

slides - NodeList

Slides NodeList instance.
Default Value: null

transInAfter - Object

Default transIn (incoming slide) "after" settings. Sets z-index:2.
Default Value: (See source)

transInBefore - Object

Default transIn (incoming slide) "before" settings. Sets visibility:visible and z-index:1.
Default Value: (See source)

transition - Slide Transition

Default slide transition.
Default Value: Slideshow.PRESETS.fade

transOutAfter - Object

Default transOut (outgoing slide) "after" settings. Sets visibility:hidden and z-index:-30000.
Default Value: (See source)

transOutBefore - Object

Default transOut (outgoing slide) "before" settings. Sets z-index:2.
Default Value: (See source)
